# Contributing to PDF2Img

Thank you for your interest in contributing to PDF2Img! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a new branch** for your feature or bug fix.
4.  **Make your changes** and commit them.
5.  **Push your changes** to your fork.
6.  **Submit a Pull Request** to the main repository.

## Development Setup

1.  Create a virtual environment:
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    ```

2.  Install dependencies in editable mode with all extras:
    ```bash
    pip install -e ".[all]"
    ```

3.  Run the tests (if available) or verify functionality manually using the CLI or UI.

## Code Style

*   Please follow standard Python coding conventions (PEP 8).
*   Ensure your code is readable and well-documented.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub.
